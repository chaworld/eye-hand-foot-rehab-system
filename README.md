# 智慧復健及眼球詮釋辨識系統 (Smart Rehabilitation and Eye-Hand-Foot Recognition System)

這是一個結合眼動追蹤、手勢辨識與足部偵測的智慧復健系統，基於 Python、MediaPipe、OpenCV、Pygame 與 CustomTkinter 開發。

## 主要功能
- **眼部辨識復健系統**：追蹤眼球運動進行復健訓練。
- **手部/足部偵測**：利用 MediaPipe 進行精確肢體動作捕捉。
- **互動遊戲界面**：結合 Pygame 提供直觀的練習體驗。

## 安裝與執行
1. 安裝依賴項目：
   ```bash
   pip install -r requirements.txt
   ```
2. 執行主程式：
   ```bash
   python app.py
   ```
