# 智慧復健及眼球詮釋辨識系統

## 安裝與執行

1. 下載專案：
   ```bash
   git clone https://github.com/chaworld/eye-hand-foot-rehab-system.git
   cd eye-hand-foot-rehab-system
   ```
2. 安裝依賴項目：
   ```bash
   pip install -r requirements.txt
   ```
3. 執行主程式：
   ```bash
   python app.py
   ```
