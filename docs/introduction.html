智慧復健及眼球詮釋辨識系統

Smart Rehab & Eye-Hand Interpretation System

1. 專案簡介 (Introduction)

本專案整合了先進的電腦視覺技術（MediaPipe），提供一套非接觸式的智慧復健與輔助溝通系統。系統主要分為三大功能模組：「眼部辨識復健系統」、「手部辨識復健系統」與「腳步辨識復健系統」，旨在協助需要肢體復健或言語溝通輔助的使用者，透過自然的眼球運動、手勢動作或腳步移動來與系統互動。

核心特色

  • 非接觸式操作：僅需使用一般攝影機（Webcam），無需昂貴的穿戴式裝置。
  • 多模態互動：支援眼球視線追蹤、眼球手勢指令、手部精細動作（抓取）偵測，以及腳步移動追蹤。
  • 即時語音回饋：動作觸發後提供語音提示，增強互動體驗與溝通效率。
  • 遊戲化復健：透過趣味性的分類遊戲（撿豆子）及步數計數，提升復健意願。

2. 功能說明 (Features)

2.1 眼部辨識復健系統

此模組專為眼球靈活度訓練及基礎溝通設計。

  • 九宮格溝通板：透過視線停留，可觸發預設的生活用語（如：吃飯、洗澡、不舒服等）。
  • 眼球手勢指令 (Gesture Control)：系統可辨識特定的眼球連續運動軌跡，快速觸發功能：
    • ↕️ 上下掃描：觸發「喝水」語音。
    • ↔️ 左右掃描：觸發「吃飯」語音。
    • 🔄 順時針轉動：觸發「洗澡」語音。
    • 🔄 逆時針轉動：觸發「上廁所」語音。

2.2 手部辨識復健系統

此模組專為手部精細動作訓練設計，特別是針對「捏取（Pinch）」動作的練習。

  • 撿豆子遊戲：畫面中會出現虛擬豆子與碗，使用者需透過手勢將豆子抓起放入碗中。
  • 手勢偵測：偵測食指與拇指的距離來判斷是否進行「抓取」。
  • 多種模式：
    • 基本模式：無限練習，計分累加。
    • 挑戰模式：包含關卡設計（區分紅綠豆）、限時挑戰與過關分數目標。

2.3 腳步辨識復健系統

此模組專為下肢活動能力訓練設計，透過攝影機追蹤腳部移動。

  • 腳部追蹤：使用 MediaPipe Pose 偵測腳踝、腳跟、腳尖等 6 個關鍵點。
  • 步數計數：自動偵測抬腳動作並計算步數。
  • 軌跡視覺化：即時顯示雙腳移動軌跡，方便觀察動作。
  • One Euro Filter：採用動態濾波技術，減少抖動同時保持靈敏度。
  • 距離無關：使用正規化座標，無視距離與立體空間限制。

3. 使用說明 (User Manual)

3.1 啟動系統

  1. 執行專案目錄下的 app.py。
  2. 在主選單畫面中，選擇欲使用的功能：
    • 點選 「眼部辨識復健系統」 按鈕進入眼控模式。
    • 點選 「手部辨識復健系統」 按鈕進入手勢遊戲模式。
    • 點選 「腳步辨識復健系統」 按鈕進入腳步追蹤模式。

3.2 眼部辨識復健系統操作

  1. 校正準備：啟動後請保持頭部穩定，直視螢幕，讓系統捕捉臉部特徵。
  2. 九宮格操作：
    • 移動視線注視螢幕上的九宮格選項。
    • 當視線停留在某個格子一段時間後，格子會變色並播放對應語音。
  3. 手勢指令操作：
    • 若要快速表達需求，請連續進行眼球運動（如連續快速左右移動眼球）。
    • 當系統偵測到規律軌跡（如連續來回 3 次）時，會自動觸發對應指令。

3.3 手部辨識復健系統操作

  1. 啟動偵測：進入介面後，點選 「▶ 啟動偵測」 按鈕開啟攝影機。
  2. 基本操作：
    • 將手掌張開面對鏡頭。
    • 移動手部，將畫面上的游標（通常為食指與拇指中心）對準「豆子」。
    • 抓取動作：將食指與拇指指尖捏合（模擬捏豆子動作）。畫面上會顯示 "GRABBING"。
    • 移動與放置：保持捏合姿勢，移動手部將豆子拖曳至「碗」的上方。
    • 放開動作：張開手指，豆子即會落下。若成功落入碗中，分數將會增加並播放音效。
  3. 切換模式：點選 「切換遊戲模式」 可挑戰更進階的關卡（如：要在限制時間內只撿綠豆，避開紅豆等）。

3.4 腳步辨識復健系統操作

  1. 啟動偵測：進入介面後，點選 「▶ 啟動偵測」 按鈕開啟攝影機。
  2. 站姿準備：
    • 站在攝影機前方，確保雙腳在畫面範圍內。
    • 系統會自動偵測腳踝、腳跟、腳尖等關鍵點。
  3. 步伐練習：
    • 進行原地踏步或抬腳動作。
    • 系統會自動偵測抬腳動作並計算步數。
    • 每次成功踏步會播放音效回饋。
  4. 軌跡觀察：畫面會即時顯示雙腳的移動軌跡（左腳橘色、右腳藍色）。
  5. 重置步數：點選 「🔄 重置步數」 可將計數歸零重新開始。

4. 專案結構與套件需求 (System Requirements)

套件列表說明：
  • customtkinter: 用於建立現代化 GUI 介面
  • pygame: 用於遊戲與多媒體互動
  • mediapipe: Google 開發的機器學習庫，用於眼球與手部追蹤
  • opencv-python: 電腦視覺影像處理
  • Pillow: 影像處理
  • numpy: 數值運算
