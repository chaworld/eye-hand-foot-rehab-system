# 安裝步驟

### 步驟 1: 開啟終端機
1.  開啟 **PowerShell** 或 **命令提示字元 (CMD)**。
2.  切換到專案目錄：
    ```powershell
    cd "你的資料夾路徑"
    ```

### 步驟 2: 建立虛擬環境 (建議，不想用就跳到步驟 3)
為了避免套件衝突，建議建立 Python 虛擬環境：

```powershell
# 1.建立虛擬環境 (名稱為 venv)
python -m venv venv

# 2.啟動虛擬環境 (Windows)
.\venv\Scripts\activate  #若是 Mac/Linux 請使用: source venv/bin/activate
```

*啟動後，您的終端機提示字元前方應該會出現 `(venv)` 字樣。*

### 步驟 3: 安裝相依套件
執行以下指令以安裝專案所需的 Python 函式庫：

```powershell
pip install -r requirements.txt
```

# 執行程式

### 方式一：使用主選單 (建議)
執行主程式 `app.py`，它提供了一個圖形化選單來啟動不同功能：

```powershell
python app.py
```

### 方式二：個別啟動模組
若您只需要特定功能，也可以直接執行對應的檔案：

*   **眼部辨識復健系統 (GUI + 九宮格)**:
    ```powershell
    python gui_module.py
    ```
*   **眼球詮釋辨識 (四功能)**:
    ```powershell
    python pygame_module.py
    ```
    **直接啟動腳步辨識**
    ```powershell
    python foot_gui.py
    ```
